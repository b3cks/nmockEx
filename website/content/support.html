Support for NMock is available through our SourceForge mailing list. NMock developers and users regularly discuss NMock features and techniques for using the library and you should find a friendly audience for your questions. To subscribe to the mailing list, please visit this page.

We use SourceForge trackers for both bugs and feature requests. Please do quickly check these trackers before posting to the mailing list, however in most cases you'll get a better response from a human being.
